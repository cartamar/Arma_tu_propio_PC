Memoria RAM

La Memoria de Acceso Aleatorio, Random Access Memory (RAM), por sus siglas en inglés. Es la parte de almacenamiento temporal del pensamiento de la computadora si la comparas con un cerebro.

La RAM almacena los datos en los que trabaja activamente la CPU. Puede leer y escribir muy rápido pero pierde todo lo que había almacenado una vez que se apaga. Una placa madre tiene ranuras de RAM donde la RAM se conecta. La RAM viene en diferentes tamaños, desde alrededor de 4 GB hasta 32 GB por Módulo en los sistemas actuales. Tener más Módulos por supuesto multiplicará la cantidad de RAM.

El estándar actual de las memorias RAM es DDR4 (Double Data Rate)  y este tienes varias frecuencias donde, entre mayor sea mejor será la Memoria RAM, sin embargo este esta limitado por el propio CPU, ya que si tienes un CPU que solo soporta memorias RAM con una velocidad de Reloj de 2600, no importa que la memoria llegue a 3000, solo podrá ir a 2600, así que  para ajustarte al procesador y no gastar de más, es recomendable buscar la frecuencia máxima que permite em memorias RAM. 

En caso de tener un Procesador con GPU integrada y utilizar la GPU de este, es necesario tener una buena cantidad de memoria RAM, ya que devido a que la GPU integrada no tiene su propia Memoria, esta utiliza la de la RAM, para esto, es mejor tener una buena cantidad de memoria RAM, ya que en actividades que necesiten video, esta será muy requerida.
